# TaskFlow

TaskFlow is a modern and responsive workflow management landing page designed to help users organize tasks and improve productivity. The website provides a clean interface with sections highlighting features such as security, analytics, teamwork, cloud access, and customization.

## Features

* Responsive design for desktop and mobile devices
* Smooth and user-friendly interface
* Hero section with call-to-action
* Feature cards with icons and images
* Dark mode toggle
* Mobile navigation menu
* Sections for Features, Pricing, and Contact
* Interactive functionality using JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript
